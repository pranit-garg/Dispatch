shortDescription
Dispatch is micropayment infrastructure for machine-to-machine commerce. AI agents pay for compute with USDC via x402 HTTP headers, settlements auto-swap to BOLT via Jupiter DEX, and 5% of every payment burns permanently. No accounts, no API keys, no humans required.
whatItDoes
Dispatch is DeFi infrastructure for the machine economy. AI agents need compute but cannot create accounts, manage API keys, or pay with credit cards. Dispatch solves this with a payment-first protocol. An agent sends one HTTP POST with an x402 payment header (USDC). That request handles discovery, payment, processing, and verification. The coordinator matches the job to the best worker, the worker runs real LLM inference, signs an ed25519 receipt over the output, and returns the result. The DeFi layer: USDC auto-swaps to BOLT via Jupiter DEX on every settlement. 5% of BOLT is burned permanently, making supply deflationary. Workers stake BOLT across three tiers: Open (standard), Verified (100 BOLT, 1.5x reputation), Sentinel (1,000 BOLT, 2x reputation plus revenue share). Every job creates DeFi activity: a Jupiter swap, a burn event, reputation feedback onchain. Dispatch turns compute demand into continuous DeFi volume.
defiIntegration
Five DeFi components in every job settlement: 1. x402 Micropayments: USDC payment in the HTTP header. No wallet connection, no approval tx, no pre-funding. Atomic with the request. Works on Solana (SPL) and Monad (ExactEvmScheme). 2. Jupiter DEX: Coordinator auto-swaps USDC to BOLT via Jupiter after each job. Every job creates real swap volume. 3. BOLT Token: Native payment token (SPL on Solana, wBOLT ERC-20 on Monad). Workers earn BOLT per job. Total supply: 1 billion, 9 decimals. 4. Deflationary Burn: 5% of every BOLT settlement burned permanently. Supply decreases with every job processed. 5. Staking Tiers: Workers lock BOLT for priority and reputation amplification. - Open: 0 BOLT, standard matching - Verified: 100 BOLT, +5 priority, 1.5x reputation - Sentinel: 1,000 BOLT, 2x reputation, revenue share Staked tokens leave circulation. Combined with burn, circulating supply shrinks from two directions.
howItWorks
End-to-end payment flow for a single compute job: 1. Agent submits job with payment. HTTP POST with x402 header (USDC amount, payer address, signature). Coordinator validates payment before routing. 2. Coordinator routes by reputation. Queries ERC-8004 Reputation Registry on Monad. Composite score: base reputation (0-100) x stake tier (Open 1x, Verified 1.5x, Sentinel 2x). Routes to highest-scoring worker. 3. Worker processes and signs. Atomic claim prevents double-processing. Real LLM inference via Ollama. Worker signs ed25519 receipt over SHA-256 output hash. 4. Settlement triggers DeFi flow. USDC auto-swaps to BOLT via Jupiter DEX. 5% protocol fee. 5% of remaining BOLT burned. Net BOLT goes to worker. 5. Reputation updates onchain. Feedback posts to ERC-8004 on Monad: score, skill tag, endpoint, feedback hash. Track record grows, improving future matching. Every step is automated. No human approves, routes, or settles. Payment and DeFi run as pure infrastructure.
